Maxwell may refer to:

  • Maxwell (NPC), a non-playable character in the game who acts as the main antagonist of the Characters and spawns at the start of each new world.
  • Maxwell (Character), a playable character in the game after he is freed from the Nightmare Throne after completing Adventure Mode.
  • William Carter, a nonplayable character in the ARG puzzles made by Klei.

                      

Maxwell however can be spawned in by using console command ("waxwell") instead of ("maxwell")

Using the command ("maxwell") will spawn in the Maxwell seen at the start of your world.

Using the code ("waxwell") will spawn the character version of Maxwell.
