The Bonfire is a removed structure that is in the files of Don't Starve. It can be spawned into the game using the command DebugSpawn("bonfire").

Use

The Bonfire automatically starts on its own at night, making a huge fire. However, the fire does not cast any light - the Bonfire used to be where Pigs were before Pig Houses were implemented. The Bonfire has a map icon that looks like a Pig House. The player can't interact with it, with the exception of cooking food over it.

Bugs

  • The game will crash upon opening the map as long as the Bonfire is still there.
