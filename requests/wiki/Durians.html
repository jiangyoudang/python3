Durian is a rare Fruit that can be Farmed from Farm Plots by planting Seeds for a 5% chance of growing or using Durian Seeds. It can be Cooked on fire into Extra Smelly Durian, and also used as an ingredient in several Crock Pot recipes. Feeding a Durian to Birds in a Birdcage will produce 1–3 Durian Seeds, and half the time 1 regular Seed.

Like Monster Meat, Durian damages health instead of restoring, which also affects its Crock Pot recipes. It can be used as a Fruit/any filler in Crock Pot recipe with no toxic effects, but using more than 1 Durian in a Crock Pot recipe could result in Monster Lasagna, unless the recipe contains Twigs.

Extra Smelly Durian

Extra Smelly Durian is a Fruit Food item made by Cooking a Durian. This cooked version does not damage health. It can be used as an ingredient in several Crock Pot recipes.

Like Cooked Monster Meat and the uncooked Durian, it will produce Monster Lasagna if more than one is included in the Crock Pot. It can be used in place of the uncooked Durian for all Crock Pot recipes.

Trivia

  • The sanity loss, character quotes, and name "Extra Smelly" refer to the foul odor that the fruit can emit. Not all varieties of Durian have a foul odor, but it can be so bad and linger for so long that Durians have been banned in some establishments.
