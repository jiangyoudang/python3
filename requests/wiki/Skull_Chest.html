Skull Chest

Tool required
DebugSpawn "skullchest"

The Skull Chest is an alternative Chest found in the game files. It can only be seen in game by spawning via the Console. As of now, it has exactly the same functionality as a chest, although most players still say it looks scary or dangerous.

The .tex file can be found in the Don't Starve game files as of version "All's Well that Maxwell".

This item also appears in the mod called Lost Fragment, a mod made by the developers which was part of the William Carter Puzzles.
