This article has been removed from the game / will not be implemented.
The information presented here is for reference only.


It was possible to import a save from a backed up region that can be found in the steam directory. Since the Doorway to Adventure Update, importing has been removed from the game. However, players can still import their saving manually by copying other game files into the directory below:

C:\ProgramData\Steam\Player\(your steam ID)\storage.

For more detailed information of saving, please read here.

Notes

Importing was only available for the steam version of the game. If you are trying to import to a specific game slot, the slot must match the original slot of the saved game.
