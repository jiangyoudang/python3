The beta preview is a function which can be enabled via the game settings in Steam. A beta preview is a preview of functions which are not implented in the game yet, where you can give feedback on.

Enabling beta previews

Go to your game library and right click on Don't Starve. Then click on Properties and go to the tab Beta's. There you can select "NONE" (no beta updates, or if you were subscribed to them, you can opt-out) or "publicpreview". There you have to select "publicpreview". Then accept the changes. After that, the beta will download. Now you can play with the beta preview!

When a preview is over, the beta automatically opts out.
