This category lists the various Structures are either crafted in-game, or sometimes found around the world. Structures are special objects that serve a passive purpose such as using a wall to defend or blockade an area, or an active purpose, such as a Houndius Shootius being used to protect an area against mobs.


Structure Radius

When placed, every structure or plantable has a radius.

If anything already exists within that radius, it can't be placed there. It doesn't take into account what's already there; it only counts the radius of the object being placed. (This is why, for example, you can place a chest very close to an existing birdcage, but if you try to place a birdcage next to an existing chest, you need to place it significantly further away.)The following are the radius values from the LUA files: 

TABLE OF MINIMUM RADIUS VALUES FROM EXISTING OBJECTS
Structure Radius Notes
Houndius Shootius 0.00 Can be placed anywhere on valid ground
Walls  0.10 Distance from other walls
Walls 1.00 Distance from things that aren't walls
Tooth Trap  0.75
Potted Fern 0.90
Chest 1.00
Bee Mine 0.75
Ice Box 1.50
All Other Plantables 2.00 Things that can be items
All Other Structures 3.20 Things that can't be items

Note on Plantables vs Structures: For the purposes of the table above, if you can build it and it goes into your inventory, it's a plantable. If you can pre-build it and place it later, it's a structure.

If looking to build stuff tightly together, you need to place the large items first. (For example, build the birdcage first, then the walls around it, not the other way around. Build the crockpot, then the chests and so forth.)

Note on Architectural Geometry: if you're running this mod, by default, snap is 0.5 units. That means you can build stuff "as close to the planned wall as possible" by hovering over the ground where the wall will go, then moving 2 clicks diagonally away, and building. When you build the wall, it will be 1 unit away, the closest it can be from the drying rack/crock pot/bird cage/etc.
