This page describes an upcoming or planned feature and may contain major spoilers.
The name and content may not be accurate and may contain speculation.

The Sunk Boat is an unimplimented/planned feature found in the game files. Currently, it can only be spawned in with the Lua Console, using the code DebugSpawn("sunkboat"). However, it will not have a texture, though it will block you if you try to walk where it was spawned. The picture is taken from the game files. Originally, it may have been how you arrived on the island, as when Wilson examines it's counterpart, the boat, he says "Is that how I got here?".
