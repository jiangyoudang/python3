“ SYSTEM OVERLOAD ”

–WX-78

Lightning occurs naturally during heavy Rain and can be very dangerous as it may start uncontrolled Fires. When it occurs, it will randomly strike an object or item on screen, setting it ablaze if it can be burned. Lightning bolts can be contained by a Lightning Rod, which draws the lightning and produces Light for roughly two days after being struck. Players can be damaged from fires started by lightning. Moreover, any Mob struck by lightning will run around spreading the fire if it runs by any flammable object. Using an Umbrella should shield the player from being struck.

Lightning will also strike the player on several occasions beside a random, natural occurrence during storms. Being resurrected with the Touch Stone will always cause the player to be struck, and similarly, using a Telelocator Staff or damaging an Ancient Pseudoscience Station will cause the same effect.

When struck by a lighting bolt, WX-78 will initiate its System Overload power, regardless of the bolt's origin. 

Reign of Giants

With the Reign of Giants DLC, lightning can strike the player, dealing 10-15 points of damage.

Volt Goat can be struck by lightning causing it to become aggressive towards the player. This allows them to drop Electric Milk.

Bugs

  • Occasionally if WX-78 dies and then is immediately struck by lightning, he will appear to be resurrected and the game will crash.
