The Tesla Tree is an object which is in development for the Reign of Giants DLC. Currently it only exists in the game files as an unassembled TEX file.

Bryce from Klei stated that this tree was originally going to appear in some sort of "Tesla Biome", this is where Volt Goats were originally going to be found. It would have been a key part in a larger system that was scrapped from development. [1]
