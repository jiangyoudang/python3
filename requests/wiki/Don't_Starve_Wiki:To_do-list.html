    Minor Checks

    Standard/quality checks that are performed by visiting editors daily. For newbie editors with little experience, this is a good place to begin to become familiar with the wikia system and how to edit.

    • Check grammar and spelling on pages.
    • Check which url links work, if they do not, note what urls are not working in the comments section.
    • Add links to and from orphaned pages so they are easier to find.
    • Check out Special:UnusedFiles, find a use for these images or put them up for deletion.
    • This needs to be regularly cleared out. This also needs to be kept clean, except for a few important videos the wikia needs to use for its own purposes. If either gets too full, point this out to a admin or bureaucrat so they can clear these.
    • Discuss changes to the Manual of Style on its talk page.
